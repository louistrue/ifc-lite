Add IFC diff and clash detection features

- New `@ifc-lite/diff` package: GlobalId-based model comparison with attribute, property, and quantity change detection
- New `@ifc-lite/clash` package: collision, clearance, and intersection detection with AABB broad-phase and triangle narrow-phase
- Enhanced `ifc-lite diff` CLI command with property-level changes and viewer visualization
- New `ifc-lite clash` CLI command with configurable modes, type filters, and JSON config support
- Viewer UI: interactive Diff and Clash panels with sorting, filtering, federation-aware selection, color overlays, and isolation
