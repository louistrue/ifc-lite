# Cum pornești aplicația

## Viewer (mod development)

```bash
pnpm dev
```

> Rulează build complet pentru toate pachetele, apoi pornește viewer-ul cu Vite.

### Alternativ, doar viewer-ul (fără rebuild):

```bash 
pnpm --filter viewer dev

```

## Alte comenzi utile

| Comandă | Descriere |
|---|---|
| `pnpm dev:embed` | Pornește embed viewer |
| `pnpm dev:desktop` | Pornește aplicația desktop (Tauri) |
| `pnpm build` | Build complet pentru toate pachetele |
| `pnpm test` | Rulează toate testele |
| `pnpm docs:serve` | Pornește serverul de documentație |
